# Resume Build Like A Pro

A modern, responsive resume builder built with React and Tailwind CSS.

## Features

- 3 Layouts: Professional, Modern, Casual
- 3 Background Themes: Blue, Beige, Grey
- Live Preview Mode
- Edit / Share / Download PDF (custom filename)
- Clean & Corporate Style
- SPA (Single Page App)
- Footer: "© 2025 Resume Builder Pro — Designed by Spandana H. P"

## Installation

1. Navigate to project folder
2. Run `npm install`
3. Run `npm start` to open locally
